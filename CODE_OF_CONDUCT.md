# Code Of Conduct

This project uses a simple contributor standard:

- Be respectful and direct.
- Keep feedback focused on the work.
- Assume good intent, but prioritize user safety and project quality.
- Do not harass, threaten, or discriminate against other contributors.

Maintainers may remove comments, issues, pull requests, or contributors that violate these expectations.

Report conduct issues privately to the maintainer when possible.
